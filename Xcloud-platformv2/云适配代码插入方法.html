WordPress安装云适配代码方法

首先从云适配IDE平台里获取安装代码，在后台文件中查找 wp-content/themes/head.php 文件，然后打开，将云适配代码复制粘贴到页面开头 < head >开标前后面,保存并更新文件。



phpcms安装云适配代码方法

只需要将header以及header_page俩个文件中插入云适配代码即可。

Discuz安装云适配代码方法

首先从云适配IDE平台里获取安装代码，然后登录discuz 后台，找到路径 \template\default\common 里边有七个header相关文件，如下图中蓝色框内的htm文件进行修改：

在header文件中直接将云适配代码插到页面的第一行， 在header_common文件中将云适配代码插到 < head > 标签下第一行。

然后进入Discuz的后台，更新页面即可。

插入代码的方法就是上边提到的直接加在 < head > 标签下第一行。 然后进入DeDecms的后台，如下图中的步骤，更新页面即可。

Web服务器配置

在Web服务器Apache上配置

<Directory>

Options IncludesNOEXEC

# AllowOverride None

AllowOverride All

Order allow,deny

allow from all

AddOutputFilter LAYOUT htm

Layout '<head>' '<head> <script id="allmobilize" charset="utf-8" src="http://ysp.www.gov.cn/52011eeb721c5f42dee83bf7ea9f6cc6/allmobilize.minjs"></script><meta http-equiv="Cache-Control" content="no-siteapp" /><link rel="alternate" media="handheld" href="#"/>' replace

LayoutFooter " "

</Directory>

在Web服务器Nginx上配置

在编译 nginx 时, 加入 --with-http_sub_module 参数

并在 nginx 的配置文件中的 server 下, 加入下面的语句

sub_filter      <head>

    '<head><script id="allmobilize" charset="utf-8" src="http://a.yunshipei.com/lenovocomcn/allmobilize.min.js"></script><meta http-equiv="Cache-Control" content="no-siteapp" /><link rel="alternate" media="handheld" href="#" />';

sub_filter_once on;
